## ğŸ†˜ Troubleshooting

### **Common Issues**

**Q: Styles not applying?**
- Ensure `<link rel="stylesheet" href="fed.css">` exists
- Check file path is correct
- Clear browser cache
- Verify CSS file is in same directory

**Q: Mobile layout broken?**
- Check viewport meta tag in HTML
- Verify media queries in fed.css
- Test at exact breakpoint widths
- Check for z-index conflicts

**Q: Buttons not hovering?**
- Verify hover CSS is present
- Check for conflicting CSS
- Ensure transition property set
- Test on different browsers

**Q: Links not working?**
- Verify href paths are correct
- Check Indeed URLs are valid
- Ensure file names match
- Test in different browsers
